# Spec Kit (Manual Setup)

This repo uses a lightweight Spec Kit layout under `.specify/`.

- `.specify/memory/constitution.md` = rules of the project
- `.specify/specs/<FEATURE>/spec.md` = requirements
- `.specify/specs/<FEATURE>/plan.md` = implementation plan
- `.specify/specs/<FEATURE>/tasks.md` = task list

Templates live in `.specify/templates/`.
